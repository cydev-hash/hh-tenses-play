🔊 How to Generate Audio Files

Audio files use Microsoft's free neural TTS voice (en-US-JennyNeural). Run this once on your PC to generate all MP3s, then commit them to GitHub.

1
Install edge-tts (one time only)
Open PowerShell or Command Prompt and run:
pip install edge-tts
2
Run the audio generator in your project folder:
cd C:\GitHub\hh-tenses-play node generate-audio.js
This creates an audio/ folder with one MP3 per word form.
3
Commit and push the audio files:
git add audio/ git commit -m "Add pronunciation audio" git push
4
Done! The app will now play natural MP3 audio. If a file is missing, it automatically falls back to the browser's built-in voice.
